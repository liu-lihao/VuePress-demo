# 自动侧边栏屏蔽文件夹规则

识别 docs 目录下文件夹（无限嵌套识别），默认屏蔽 .vuepress 、 assets 。

如需修改，打开：/docs/.vuepress/tool/autoSidebar.js 。

# 修改侧边栏目录连接符

默认 - ，如： 文件夹分组 - 嵌套文件夹分组 。

如需修改，打开：/docs/.vuepress/tool/autoSidebar.js 。
